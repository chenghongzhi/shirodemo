## 主要功能就是关于文件的上传、下载，根据不同的角色分不同的权限。现在一共有3个角色，老师、学生、管理员。本项目技术点没什么，可以作为一个springboot+shiro+mybatis的一个练习项目，可以基于本项目继续开发当一个毕业设计什么的(手动滑稽)
